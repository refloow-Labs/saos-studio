# SAOS Studio — Bundle Info

A single-folder snapshot of everything important about SAOS Studio: brand assets, project overview, strategy docs, and legal templates. Use this as the canonical reference when onboarding collaborators, briefing tools, or rebuilding context.

| Folder / File | Purpose |

| `BRAND_GUIDE.md` | Logo usage, colors, typography, voice & tone |
| `logos/` | Primary dark and white logos (PNG) |
| `screenshots/` | Hero shots and product screenshots of the website + Core Service CRM |
| `legal-templates/` | Service Agreement, NDA, Privacy Policy, Terms, Proposal, Invoice, SOW, Onboarding Checklist (HTML) |
| `MARKETING_STRATEGY.md` | Positioning, ICP, channels, 90-day roadmap |

| `LAUNCH_READINESS.md` | Launch checklist and cofounder reflection |
