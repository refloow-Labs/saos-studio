2 — Mobile strategy

Three.js shaders are heavy on phones. How precious is the 3D experience on mobile?

A

Full 3D everywhere

Mobile gets the same shader + wordmark. Reduced quality (lower DPR, simpler shader, lower poly mesh) but the journey is identical.

Risk — slow phones, battery drain

B

Reduced 3D on mobile

Shader background runs at half resolution; wordmark sculpture stays but doesn't morph through every state — it just rotates and recolors. Particle/smoke states fall back to elegant 2D treatments.

Sensible · still impressive

C

2D fallback on mobile

Mobile gets a refined 2D version — the current site polished, with CSS gradients standing in for the shader and the existing logo image instead of 3D. Desktop alone gets the 3D experience.

Safest performance · two designs to maintain
